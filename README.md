# shadbook

A [storybook](https://storybook.js.org/) for [shadcn/ui](https://ui.shadcn.com/), [tailwind](https://tailwindcss.com/), and [tailwind-css-animate](https://github.com/jamiebuilds/tailwindcss-animate).

An educational project.

## How to run

```bash
npm install
npm run storybook
```
