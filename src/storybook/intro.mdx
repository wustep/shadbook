# Introduction

Welcome to <b>shadbook</b>! This site is an interactive storybook playground for [shadcn/ui](https://ui.shadcn.com/), [tailwind](https://tailwindcss.com/), and [tailwindcss-animate](https://github.com/jamiebuilds/tailwindcss-animate).

You can also use this repo as a starting point for your own project's design system, or simply use this storybook as a reference for how to use these libraries.

## Using this repo

<Unstyled>
	<Button size="sm" className="mt-2">
		<a href="https://github.com/wustep/shadbook">View this repo on GitHub</a>
	</Button>
</Unstyled>

To get started, you can clone the repository and run the storybook server:

```bash
git clone https://github.com/wustep/shadbook.git
npm install
npm run storybook
```

Check out the [README](https://github.com/wustep/shadbook/blob/main/README.md) for more info!
