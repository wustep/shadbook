# Introduction

Welcome to <b>shadbook</b>! This site is a storybook playground for shadcn/ui, tailwindcss, and tailwind-css-animate.

You can also use this repo as a starting point for your own project.

## Using this repo

View this repo on [GitHub](https://github.com/wustep/shadbook).

To get started, you can clone the repository and run the development server:

```bash
git clone https://github.com/wustep/shadbook.git
```

```bash
npm install
```

```bash
npm run storybook
```
